Branch: directs computer to go to another part of the program Clone: hardware or software that is designed in the same way as another system Commit: last step in the completion of a database change Fetch: retrieval of data from a software, script, or hardware device GIT: software used to track changes to a file set; allows for coordination among programmers to develop collaboratively Github: social networking site for developers Merge: combining different versions of a file or folder Merge Conflict: occurs when Git is unable to resolve code differences between two commits automatically Push: server initiates an information transaction Pull: client requests information transmission Remote: device that allows the user to access a device or network Repository: central place where data aggregation is kept and maintained
